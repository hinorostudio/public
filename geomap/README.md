# GeoMaps

Application mobile de cartographie : parcours GPS, cartes hors ligne, import/export GPX, etc.

Ce dépôt héberge le code source du projet.  
**Site / dépôt :** <https://github.com/VOTRE_ORG/GeoMaps>

## Intégration Strava

GeoMaps peut, sur **appareil mobile** (iOS / Android), permettre à l’utilisateur de :

- se connecter à son compte Strava via OAuth ;
- envoyer un parcours enregistré vers Strava sous forme de fichier GPX (upload API).

L’authentification utilise le schéma d’URL personnalisé `geomaps://` pour le retour OAuth après autorisation sur Strava.

**Aucune donnée Strava n’est stockée sur ce site GitHub** : les jetons d’accès restent sur l’appareil de l’utilisateur (stockage sécurisé côté application).
